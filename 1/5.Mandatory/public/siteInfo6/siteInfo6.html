CRUD api

Title description, Sep 2, 2017

An CRUD api helps us take care of how we manage data. With CRUD we are able to create, read, update, and delete objects. We could be wanting to get all the data of an user in a Json array, by utilizing express to create a http request that gets the user by his id.
