How to start working with Node.js

Title description, 24-March-21

To work with Node.js it is required that you download and install the development platform through their homepage. Go to: https://nodejs.org/ and choose the one fitting for your OS.

After you have installed Node.js you are ready and set to begin creating your development environment, with a great variety of options to routing, data handling and even an easy and simple way of managing dependencies needed for your project.

Start by creating an empty folder. This is going to your project, so give it a recognizable name.

Now you can start your editor or just the terminal to start building the environment. It really doesn’t matter which way you go, but for starters the editor will show what is created in a more user friendly way.

Open the file with the editor and head to the terminal. In here you want to make sure you are in the project directory.

Create a package.js file and an app.js file. These files are the fundamentals of the webserver.

Once created, you need to install “Express”.
